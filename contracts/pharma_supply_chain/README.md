# Pharma Supply Chain Contract

Soroban contract for pharmaceutical batch traceability, cold-chain telemetry, shipment tracking, compliance checks, and inventory planning.

## Features

- Manufacturer and medication registration
- Batch creation with authenticity verification
- Shipment lifecycle with IoT-style condition updates
- Automatic cold-chain compliance tracking
- Inventory snapshots and reorder recommendations

## Testing

```bash
cargo test -p pharma_supply_chain --features testutils
```

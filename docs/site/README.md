# Stellar Uzima Contract Documentation

Stellar Uzima is a decentralized smart contract system for secure, encrypted, and role-based management of medical records on the Stellar blockchain using Soroban and Rust.

## Key Contracts

- **medical_records** — Encrypted on-chain medical record storage with RBAC
- **healthcare_payment** — Insurance claim processing and payment routing
- **appointment_booking_escrow** — Appointment scheduling with escrow-backed payments
- **identity_registry** — W3C DID-based identity management
- **escrow** — General-purpose escrow with pull-payment pattern

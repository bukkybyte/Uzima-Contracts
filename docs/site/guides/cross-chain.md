# Cross-Chain Interactions

See [docs/CROSS_CHAIN_INTERACTION_FLOW.md](../../CROSS_CHAIN_INTERACTION_FLOW.md) and [docs/cross_chain_architecture.md](../../cross_chain_architecture.md).

## Contracts

- `cross_chain_bridge` — Bridge assets and data between chains
- `cross_chain_access` — Cross-chain access control
- `cross_chain_identity` — Identity federation across chains
- `cross_chain_enhancements` — Additional cross-chain utilities

## Use Cases

- Sync medical records across hospital networks on different chains
- Cross-chain insurance verification
- Multi-chain identity federation for international patients
